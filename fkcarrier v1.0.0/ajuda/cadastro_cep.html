Importante

    Caso seja necessário alterar esta tabela, deve ser seguida rigorosamente a formatação dos dados conforme descrito abaixo:
    • Utilizar ":" como separador entre o CEP inicial e o CEP final no intervalo de CEP.
      Ex: 11111111:22222222
    • Utilizar "/" como separador entre os intervalos de CEP.
      Ex: 11111111:22222222/33333333:44444444
