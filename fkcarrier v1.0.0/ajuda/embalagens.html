Embalagens Padrão

Objetivo

    Definir as dimensões, peso e custo das embalagens que serão utilizadas para envio dos produtos.

Opção Adicionar Nova Embalagem

    Clique nesta opção para adicionar uma nova embalagem.

Descrição

    Informe uma descrição para a embalagem.

Comprimento

    Informe o comprimento em cm da embalagem.

Altura

    Informe a altura em cm da embalagem.

Largura

    Informe a largura em cm da embalagem.

Peso

    Informe o peso em kg da embalagem.

Preço de Custo

    Informe o preço de custo da embalagem, caso queira que este valor seja somado ao valor do frete.

Ativo

    Marque esta opção para o módulo considerar a embalagem como disponível.

Excluir

    Marque esta opção se desejar excluir a embalagem. Após marcado deve ser clicada a opção Excluir Embalagens Selecionadas.

Botão Salvar

    Clique no botão para validar as alterações efetuadas.

Opção Excluir Embalagens Selecionadas

    Clique nesta opção para excluir as embalagens marcadas no campo Excluir.
