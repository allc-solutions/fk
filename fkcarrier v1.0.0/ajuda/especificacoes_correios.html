Especificações Correios

Objetivo

    Definir as especificações utilizadas pelos serviços dos Correios para cálculo do frete.
    Na instalação do módulo estes dados são cadastrados automaticamente tomando como base os chamados Serviços de Varejo.

Abrindo as opções de configuração

    Clique no respectivo serviço para as opções de configuração serem mostradas conforme descritas abaixo:

Código serviço

    Informe o código associado ao serviço dos Correios em questão.
    Se possui contrato com os Correios informe o código do serviço constante no contrato.

Código administrativo

    Informe o Código Administrativo caso tenha contrato de serviço com os Correios.
    Importante: Obrigatório para os serviços E-SEDEX e PAC-GF

Senha

    Informe a Senha caso tenha contrato de serviço com os Correios.
    Importante: Obrigatório para os serviços E-SEDEX e PAC-GF

Comprimento mínimo

    Informe em centímetros, o comprimento mínimo aceito pelos Correios para a embalagem utilizada no envio dos produtos.

Comprimento máximo

    Informe em centímetros, o comprimento máximo aceito pelos Correios para a embalagem utilizada no envio dos produtos.

Largura mínima

    Informe em centímetros, a largura mínima aceita pelos Correios para a embalagem utilizada no envio dos produtos.

Largura máxima

    Informe em centímetros, a largura máxima aceita pelos Correios para a embalagem utilizada no envio dos produtos.

Altura mínima

    Informe em centímetros, a altura mínima aceita pelos Correios para a embalagem utilizada no envio dos produtos.

Altura máxima

    Informe em centímetros, a altura máxima aceita pelos Correios para a embalagem utilizada no envio dos produtos.

Somatória dimensões

    Informe em centímetros, a somatória máxima (comprimento + largura + altura) aceita pelos Correios para a embalagem utilizada no envio dos produtos.

Peso máximo - Estadual

    Informe em kilogramas, o peso máximo permitido pelos Correios para entregas dentro do mesmo Estado.

Peso máximo - Nacional

    Informe em kilogramas, o peso máximo permitido pelos Correios para entregas para outros Estados.

Intervalo de pesos - Estadual

    Informe o intervalo de pesos que os Correios utilizam para composição do valor do frete dentro do mesmo Estado.
    Caso seja necessário alterar esta tabela, deve ser seguida rigorosamente a formatação dos campos com a separação dos pesos com o caracter "/".

Intervalo de pesos - Nacional

    Informe o intervalo de pesos que os Correios utilizam para composição do valor do frete para outros Estados.
    Caso seja necessário alterar esta tabela, deve ser seguida rigorosamente a formatação dos campos com a separação dos pesos com o caracter "/".

Cubagem max isenta

    Informe a cubagem (cm³) máxima permitida pelos Correios para isenção da cobrança considerando a cubagem dos produtos.

Cubagem base cálculo

    Informe a cubagem (cm³) utilizada pelos Correios para cálculo do chamado "Peso Cúbico".

Valor Mão Própria

    Informe em Reais o valor cobrado pelos Correios para o serviço de Entrega em Mãos.

Valor Aviso Recebimento

    Informe em Reais o valor cobrado pelos Correios para o serviço de Aviso de Recebimento.

Percentual Valor Declarado

    Informe o percentual em relação ao valor do pedido que será cobrado pelos Correios para o serviço Valor Declarado.

Máximo Valor Declarado

    Informe o valor máximo garantido pelos Correios para o serviço Valor Declarado.

Seguro Automático

    Informe o valor do seguro automático dos Correios. Este é o valor ressarcido pelos Correios, caso haja extravio da entrega, quando não contratado o serviço Valor Declarado.

Botão Salvar

    Clique no botão para validar as alterações efetuadas.
