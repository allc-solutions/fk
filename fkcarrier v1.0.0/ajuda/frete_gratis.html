Frete Grátis

Objetivo

    Definir as condições de frete grátis oferecidas aos cliente.

Opção Adicionar Região

    Clique nesta opção para adicionar uma nova região.

Abrindo as opções de configuração

    Após incluir a região, clique na respectiva região para as opções de configuração serem mostradas conforme descritas abaixo:

Nome região

    Informe o nome da região.

Região ativa

    Marque esta opção para informar que a região ficará disponível.

Estados atendidos

    Marque os Estados que serão atendidos pelo frete grátis. Utilize os botões Marcar todos ou Desmarcar todos para marcar ou desmarcar todos os estados respectivamente.

Intervalo CEPs atendidos

    Neste campo podem ser informados os intervalos de CEP em que o frete grátis ficará disponível aos clientes.
    Informe o CEP inicial e o CEP final e clique no botão incluir. Após isso os intervalos de CEP serão mostrados no campo logo abaixo em formatação especial.
    Caso necessário, este campo pode ser editado manualmente, no entanto deve ser tomado cuidado com sua formatação.

Valor pedido

    Informe neste campo a partir de que valor do pedido o cliente poderá ter frete grátis.
    Importante: valor zero indica que a região não será selecionada de acordo com o valor do pedido.

Produtos

    Inclua os IDs dos produtos que terão frete grátis na região.
    Para criar a tabela de produtos com frete grátis, digite seu ID no campo apropriado e clique no botão Incluir.

Transportadora

    Selecione a transportadora que será utilizada para envio de pedidos com frete grátis.

Botão Salvar

    Clique no botão para validar as alterações efetuadas.

Excluir região

    Marque esta opção se deseja excluir a região da transportadora. Após marcado deve ser clicada a opção Excluir Regiões Selecionadas.
