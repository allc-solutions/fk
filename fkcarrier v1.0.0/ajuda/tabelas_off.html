Tabelas offline

Objetivo

    Gerar as tabelas de preço offline que serão utilizadas para cálculo do frete no caso de problemas com o web services dos Correios ou se selecionada a opção de cálculo somente offline.

Importante

    Utilize o navegador Firefox para gerar as tabelas. Os demais navegadores não atualizam a tela durante o processamento, o que pode dar a falsa impressão de que o processamento está com problemas.
    Pode ocorrer do navegador enviar mensagem informando que a página parou de responder, deve ser respondido para continuar aguardando.

Abrindo as opções de configuração

    Clique no serviço dos Correios para que as opções de configuração sejam mostradas conforme descritas abaixo:

Opção Processar todos

    Clique para que todas as tabelas referentes ao serviço dos Correios em questão sejam criadas.
    Ao término da criação de cada tabela os valores serão mostrados.

Opção Processar esta tabela

    Clique para que a tabela em questão seja criada.
    Ao término da criação da tabela os valores serão mostrados.
    Esta opção deve ser utilizada para refazer determinada tabela quando ocorrer erro na atualização geral ou por qualquer razão seja necessário gerar uma única tabela.

Opção Limpar tabelas

    Clique para que todas as tabelas do referido serviço dos Correios sejam limpas e consequentemente não utilizadas pelo módulo.

Botão Salvar

    Clique no botão para validar as alterações efetuadas.
