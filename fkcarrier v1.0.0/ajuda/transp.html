Transportadoras

Objetivo

    Definir as transportadoras disponibilizadas aos cliente, suas respectivas regiões e tabelas de preços.

Opção Adicionar Transportadora

    Clique nesta opção para adicionar uma nova transportadora.

Abrindo as opções de configuração

    Após incluir a transportadora, clique na respectiva transportadora para as opções de configuração serem mostradas conforme descritas abaixo:

Nome transportadora

    Informe o nome da transportadora. Esta informação será visível ao cliente.

Transportadora ativa

    Marque esta opção para informar que a transportadora ficará disponível aos clientes.

Grade velocidade

    Este campo informa ao Prestashop a velocidade de entrega do serviço. O valor pode variar de 0 a 9, sendo que o valor 0 indica o de menor velocidade.

Logo

    Clique no botão Selecionar arquivos para selecionar o logotipo para a transportadora. Somente são permitidos arquivos com extensão jpg.

    A imagem somente será mostrada/alterada após clique no botão Salvar.

Abrindo as opções de configuração das regiões

    Clique na respectiva região para as opções de configuração serem mostradas conforme descritas abaixo:

Nome região

    Informe o nome da região.

Prazo entrega específico

    Preencha este campo se desejar que o prazo de entrega desta transportadora seja diferente dos definidos no item 4. Prazos de Entrega.
    Este campo deve ser preenchido da forma que deve ser mostrado aos clientes, ex: Até 2 dias úteis.

Estados atendidos

    Marque os Estados que serão atendidos pelo serviço. Utilize os botões Marcar todos ou Desmarcar todos para marcar ou desmarcar todos os estados respectivamente.

Intervalo CEPs atendidos

    Neste campo podem ser informados os intervalos de CEP em que o serviço ficará disponível aos clientes.

    Informe o CEP inicial e o CEP final e clique no botão incluir. Após isso os intervalos de CEP serão mostrados no campo logo abaixo em formatação especial.

    Caso necessário, este campo pode ser editado manualmente, no entanto deve ser tomado cuidado com sua formatação.

Cobrança do frete

    Marque a forma de cobrança utilizada pela transportadora, sendo que as opções de configuração depende da forma selecionada.

Preço fixo

    A transportadora irá cobrar um valor fixo, não importando as características do produto.
    • Informe o valor fixo a ser cobrado no campo apropriado, logo abaixo.

Preço fixo por intervalo de peso

    A transportadora irá cobrar um valor fixo baseado em uma tabela de intervalos de peso, ex: até 10kg cobrar R$25,00, de 11kg a 20kg cobrar R$30,00, etc.
    Para definir a tabela informe nos campos apropriados, logo abaixo, o peso e valor cobrado e clique no botão Incluir.
    Importante: a inclusão dos dados deve ser em order crescente de peso, ex: até 10kg, até 20kg, etc.
    • Valor kilo excedente: informe o valor cobrado pelo excedente de peso, considerando o maior peso estipulado na tabela.
    • Percentual seguro: informe o percentual adicionado ao valor do frete para o seguro dos produtos. Este percentual será calculado com base na somatória dos valores dos produtos.
    • Valor pedágio: valor a ser adicionado ao frete referente ao pedágio.
    • Fator cubagem: fator de multiplicação utilizado pela transportadora para cálculo do "Peso cúbico". Normalmente as transportadoras consideram o peso e a cubagem para cálculo do frete, sendo cobrado o que for maior.

Preço peso x valor kilo por intervalo de peso

    A transportadora irá cobrar um valor por kilo baseado em uma tabela de intervalos de peso, ex: até 10kg cobrar R$5,00 por kilo, de 11kg a 20kg cobrar R$9,00 por kilo, etc.
    Para definir a tabela informe nos campos apropriados, logo abaixo, o peso e valor cobrado e clique no botão Incluir.
    Importante: a inclusão dos dados deve ser em order crescente de peso, ex: até 10kg, até 20kg, etc.
    • Valor kilo excedente: informe o valor cobrado pelo excedente de peso, considerando o maior peso estipulado na tabela.
    • Percentual seguro: informe o percentual adicionado ao valor do frete para o seguro dos produtos. Este percentual será calculado com base na somatória dos valores dos produtos.
    • Valor pedágio: valor a ser adicionado ao frete referente ao pedágio.
    • Fator cubagem: fator de multiplicação utilizado pela transportadora para cálculo do "Peso cúbico". Normalmente as transportadoras consideram o peso e a cubagem para cálculo do frete, sendo cobrado o que for maior.

Excluir região

    Marque esta opção se deseja excluir a região da transportadora. Após marcado deve ser clicada a opção Excluir Regiões Selecionadas.

Botão Salvar

    Clique no botão para validar as alterações efetuadas.

Botão Adicionar nova região

    Clique no botão para adicionar uma nova região à transportadora.

Botão Excluir transportadora

    Clique no botão para excluir a transportadora e todas suas regiões.

Botão Excluir regiões selecionadas

    Clique no botão para excluir as regiões que foram marcadas no campo Excluir região.
